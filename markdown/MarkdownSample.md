Markdown  是一個非常方便用來寫部落格的語法。可以透過一些定義好的符號或格式，便可輕易的編排你的文章，先不需要使用大量的HTML，雖然最後仍是需要轉換成HTML。先讓我們先下面的範例：

    > ### This is a header
    > 
    > 1. This is the first list item.
    > 2. This is the secord list item.

如此所產生的結果如下：

> ### This is a header
> 
> 1. This is the first list item.
> 2. This is the secord list item.
